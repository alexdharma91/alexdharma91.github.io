validation-plugin

Features

    Validation-plugin supports following types of validation:

  • Type
    • isInt
    • isFloat
  • Length
    • minLength
    • maxLength
  • is
    • isEmail
    • isUrl
    • isNotEmpty

DEMO

data-validation="isInt"
data-validation="isFloat"
data-validation="isEmail"
data-validation="isUrl"
data-validation="isNotEmpty"
